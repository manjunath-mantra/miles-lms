## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

```
## MIGRATION 
```bash
## Generate migration
npm run migrate:generate <MigrationName>

## Run migration
npm run migrate:run

## Revert migration
npm run migrate:revert

```
